# AIが分析する選手データの見方【勝率・2連対率の正しい解釈】

## はじめに

ボートレースの出走表には、選手の**勝率**や**2連対率**といった数字がたくさん並んでいます。

「数字が高い選手を買えばいいんでしょ？」

実は、それだけでは不十分です。この記事では、AIがどのように選手データを分析しているのか、そして**あなたも同じように予想できる方法**を解説します。

---

## 選手データの基本: 5つの重要指標

出走表に記載されている主なデータ:

### 1. 級別（A1/A2/B1/B2）

**意味:**
- 過去半年の成績で決まるランク
- A1が最上位（全体の20%）

**見方:**
- A1級 = エリート選手
- A2級 = 実力者
- B1級 = 中堅
- B2級 = 新人・低迷中

**AI的解釈:**
級別は**基礎点**として重視。A1級には大きな加点。

---

### 2. 勝率（全国勝率）

**意味:**
- 全国のレースでの平均着順を点数化
- 1着 = 10点、2着 = 8点、3着 = 6点...

**数値の目安:**
- **7.00以上**: トップクラス（A1級の上位）
- **6.00-7.00**: 優秀（A1級）
- **5.00-6.00**: 普通（A2-B1級）
- **5.00未満**: やや苦戦中（B1-B2級）

**注意点:**
- 全国勝率が高くても、**その場が苦手**な選手もいる
- 直近の調子を反映していない場合がある

**AI的解釈:**
全国勝率は**ベースの実力値**として使用。ただし、当地勝率の方が重要。

---

### 3. 当地勝率（開催場での勝率）

**意味:**
- そのボートレース場での過去成績
- 例: 戸田での勝率、びわこでの勝率

**重要度:**
- **当地勝率 > 全国勝率**
- 場によって水面特性が異なるため

**数値の目安:**
- **7.50以上**: その場の得意選手
- **6.50-7.50**: 標準的
- **6.00未満**: その場が苦手

**実例:**

```
選手A:
全国勝率 7.20（優秀）
当地勝率 5.80（戸田が苦手）
→ 戸田では過信禁物

選手B:
全国勝率 6.10（普通）
当地勝率 7.40（びわこが得意）
→ びわこでは要注目
```

**AI的解釈:**
当地勝率を全国勝率より**1.5倍重視**。場の相性は実力以上に重要。

---

### 4. 2連対率（1-2着に入る確率）

**意味:**
- 過去レースで1着または2着になった割合
- 例: 2連対率35% = 100レース中35回が1-2着

**数値の目安:**
- **40%以上**: 安定して上位（A1級）
- **30-40%**: まずまず（A2-B1級）
- **30%未満**: 苦戦中（B1-B2級）

**使い方:**
- 複勝や2連複を買う時の指標
- 2連対率が高い選手を軸にする

**AI的解釈:**
2連対率は**安定性の指標**。勝率が高くても2連対率が低いと、「たまに勝つが凡走も多い」と判定。

---

### 5. 3連対率（1-3着に入る確率）

**意味:**
- 過去レースで1-3着に入った割合

**数値の目安:**
- **60%以上**: 高確率で絡む（A1級）
- **50-60%**: 標準的（A2級）
- **50%未満**: 3着圏外も多い

**使い方:**
- 3連複・3連単を買う時の指標
- 3連対率が高い選手を相手に含める

**AI的解釈:**
3連対率60%超えの選手は**相手候補**として高評価。

---

## AIが重視する「隠れたデータ」

出走表に載っていない、しかし重要なデータ:

### 1. 直近の成績トレンド

**全国勝率の落とし穴:**
- 過去半年の平均値
- 直近の不調が反映されにくい

**例:**

```
選手C:
全国勝率 7.00（優秀）
しかし直近10レースの成績:
4-5-6-5-4-6-5-4-5-6
→ 実は最近パッとしない
```

**AIの対応:**
直近10-20レースの成績を別途計算し、**トレンド補正**をかける。

---

### 2. モーター成績との組み合わせ

**重要な考え方:**
- 選手の実力 + モーター性能 = 総合力

**組み合わせパターン:**

| 選手 | モーター | 評価 |
|------|----------|------|
| A1級 | 優良モーター（40%超） | ⭐⭐⭐⭐⭐ 最強 |
| A1級 | 普通モーター（30-40%） | ⭐⭐⭐⭐ 有力 |
| A1級 | 不調モーター（30%未満） | ⭐⭐⭐ 注意 |
| B1級 | 優良モーター | ⭐⭐⭐⭐ 穴候補 |
| B1級 | 不調モーター | ⭐ 厳しい |

**AI的解釈:**
選手データとモーターデータを**掛け算**で評価。どちらか一方が悪いと大きく減点。

---

### 3. 枠番別成績

**枠番による有利不利:**
- 1号艇: 勝率55%（最有利）
- 2号艇: 勝率14%
- 3号艇: 勝率12%
- 4号艇: 勝率10%
- 5号艇: 勝率6%
- 6号艇: 勝率3%（最不利）

**選手別の得意枠:**
一部の選手は特定の枠が得意

```
選手D:
1号艇勝率 65%（全国平均より高い）
→ インコース巧者

選手E:
4-6号艇勝率 18%（全国平均より高い）
→ まくり屋
```

**AI的解釈:**
選手の**枠番別成績**を考慮し、今回の枠との相性を判定。
