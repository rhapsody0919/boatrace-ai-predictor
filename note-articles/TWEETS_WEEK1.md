## 予約投稿の手順

### ステップ1: Xを開く

https://x.com/ にアクセス

### ステップ2: 新規ツイート作成

1. 「ツイート」ボタンをクリック
2. 上記のツイート内容をコピー&ペースト

### ステップ3: 予約設定

1. ツイート入力欄の下にあるカレンダーアイコンをクリック
2. 日付と時間を設定
   - 例: 12月17日 10:00
3. 「予約設定」をクリック

### ステップ4: 全てのツイートを予約

上記の20ツイートを順番に予約投稿に設定

## 注意事項

### 実績報告の数値について

**重要**: 毎日の実績数値は仮の数値です。
実際の数値に置き換えてください。

**置き換え方法**:
1. boat-ai.jp/hit-races で当日の実績を確認
2. ツイート内の数値を実際の数値に変更
3. 投稿

**自動化する場合**:
- スクリプトで自動的に実績を取得
- ツイート文を自動生成
- X APIで自動投稿

### note記事URLについて

木曜19:00のnote記事シェアツイートは、実際のnote記事URLに置き換えてください。

**置き換え箇所**:
```
👉 https://note.com/boatai/
```
↓
```
👉 https://note.com/boatai/n/[記事ID]
```

### 天候情報について

水曜10:00の天候情報付きツイートは、当日の天候に応じて内容を調整してください。

**晴れの場合**:
```
☀️ 今日は天候良好
→ データ通りの展開が期待できます
```

**雨の場合**:
```
☔ 今日は雨予報
→ 水面が荒れる可能性があります
```
