## 前提: なぜ月5万円なのか？

### 月5万円の価値

```
年間60万円の副収入
= 飲み代、趣味代が自由に使える
= 家族旅行に年2回行ける
= 老後資金に毎月積み立てられる
```

**重要なのは「再現性」:**
- 月100万円を1回稼ぐ < 月5万円を12ヶ月稼ぐ
- 安定収入の方が人生を豊かにします

### なぜ競艇なのか？

**競艇が副業に向いている理由:**

1. **少額から始められる**
   - 最低100円から購入可能
   - 株やFXのように大きな資金不要

2. **結果がすぐ出る**
   - 1レース約2分で決着
   - その日のうちに収支が確定

3. **データが豊富**
   - 選手、モーター、過去レースのデータが公開
   - AIで分析可能

4. **開催が毎日ある**
   - 平日も土日も開催
   - 自分の都合に合わせて参加できる

## Step 3: 収益化（4ヶ月目以降）

### 目標: 月+5万円

ここから本格的に収益化を目指します。

### やること

**1. 投資額を増やす（慎重に）**

**4ヶ月目の購入配分:**
```
複勝: 5レース × 200円 = 1,000円/日
3連複: 3レース × 300円 = 900円/日
3連単: 2レース × 100円 = 200円/日
合計: 2,100円/日
```

**重要:**
いきなり投資額を増やさず、**資金の範囲内で段階的に**増やします。

**2. 3連単を少額で追加**

3連単は的中率が低い（5.6%）ですが、当たれば大きいです。

**3連単を買うレースの条件:**
- AIスコアが1位と2位で3900以上
- 1号艇A1級 × モーター2連対率45%以上
- 静水面 × 風速2m以下
- 12Rの優勝戦級レース

**3. 高配当レースを見極める**

**高配当の特徴:**
```
【狙い目】
- 2-4号艇がA級選手 × 優良モーター
- 1号艇がB級 または モーター不良
- 追い風3m以上（外側有利）

【避ける】
- 1号艇A1級 × モーター2連対率50%超
→ 堅すぎてオッズが低い
```

### 4ヶ月目の期待成績

```
【複勝】
投資: 30,000円（200円×5レース×30日）
払戻: 33,930円（260円×52.1%×150レース×平均配当補正）
収支: +3,930円

【3連複】
投資: 27,000円（300円×3レース×30日）
払戻: 56,187円（平均配当2,080円×15.3%×90レース×3倍投資）
収支: +29,187円

【3連単】
投資: 6,000円（100円×2レース×30日）
払戻: 26,982円（平均配当24,188円×5.6%×60レース）
収支: +20,982円

合計: +54,099円
```

**月+5万円達成！**

## 失敗パターンと対策

### 失敗パターン1: 調子に乗って投資額を増やす

**NGパターン:**
```
3連単が2回連続で的中
→ 「今日は調子がいい！」
→ 次のレースで1,000円購入
→ 外れて利益を全て失う
```

**対策:**
どんなに調子が良くても、**購入額は変えない**

### 失敗パターン2: 負けを取り返そうとする

**NGパターン:**
```
5レース連続で外れた
→ 「次こそは当てて取り返す！」
→ 倍額購入
→ さらに外れて赤字が拡大
```

**対策:**
負けた日は**そこで終了**。翌日また同じ額から始める。

### 失敗パターン3: 高配当ばかり狙う

**NGパターン:**
```
3連単で万舟券を狙う
→ 的中率5.6%なので18回に1回しか当たらない
→ その間の17回で資金が尽きる
```

**対策:**
**複勝7割、3連複2割、3連単1割**のバランスを守る
