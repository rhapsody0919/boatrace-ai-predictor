## 1. noteアカウント作成

### ステップ1: noteにアクセス

ブラウザで https://note.com/ を開きます。

### ステップ2: 会員登録

1. 右上の「新規登録」ボタンをクリック
2. 登録方法を選択:
   - **メールアドレス**で登録（推奨）
   - Googleアカウントで登録
   - Twitterアカウントで登録
   - Appleアカウントで登録

### ステップ3: アカウント情報を入力

**メールアドレスで登録する場合**:

1. メールアドレスを入力
2. パスワードを設定（8文字以上）
3. 「利用規約に同意して登録」をクリック
4. 確認メールが届く
5. メール内のリンクをクリックして認証完了

### ステップ4: ニックネームとnote IDを設定

1. **ニックネーム**: 表示名（あとで変更可能）
   - 例: `BoatAI` または `競艇AI予想BoatAI`

2. **note ID**: URLに使われるID（あとで変更不可）
   - 例: `boatai` または `boatrace-ai`
   - シンプルで覚えやすいものを推奨
   - 英数字とハイフンのみ使用可能

**おすすめのID**:
- `boatai` ← シンプル
- `boat-ai` ← ハイフン入り
- `boatrace-ai` ← わかりやすい

3. 「登録する」をクリック

## 2. プロフィール設定

アカウント作成後、プロフィールを充実させましょう。

### ステップ1: 設定画面を開く

1. 右上のアイコンをクリック
2. 「アカウント設定」を選択
3. 「プロフィール」タブを開く

### ステップ2: プロフィール画像（アイコン）を設定

**推奨画像**:
- サイズ: 正方形（推奨: 512×512px以上）
- 形式: JPG、PNG
- 内容: BoatAIのロゴ、または競艇に関連する画像

**画像がない場合**:
- Canvaなどの無料ツールで作成
- または、テキストベースのシンプルなロゴ
- 例: 青い背景に「BoatAI」の文字

**アップロード方法**:
1. 「プロフィール画像」の「変更」をクリック
2. 画像ファイルを選択
3. トリミングして保存

### ステップ3: ヘッダー画像を設定（オプション）

**推奨画像**:
- サイズ: 横長（推奨: 1500×500px）
- 形式: JPG、PNG
- 内容: 競艇場の写真、ボートレースのイメージ

**ヘッダー画像がない場合**:
- 初期は設定しなくてもOK
- あとで追加可能

### ステップ4: 自己紹介文を記入

**プロフィール欄（自己紹介）**:

以下のテンプレートを参考にしてください：

```
🏁 競艇AI予想「BoatAI」の公式noteです。

AIによる競艇予想を無料で提供しています。
毎日132レースの予想を自動生成し、的中率を完全公開。

✅ 複勝的中率50%以上
✅ データ分析に基づく客観的な予想
✅ 全24競艇場対応

▼BoatAI公式サイト
https://boat-ai.jp/

このnoteでは、競艇データ分析の方法、AI予想の仕組み、的中実績レポートなどを発信しています。

競艇初心者から上級者まで、予想の参考にしてください。

---
免責事項: 予想情報の提供を目的としており、的中を保証するものではありません。舟券の購入は自己責任でお願いします。
```

**ポイント**:
- 簡潔でわかりやすく
- BoatAIの特徴を明記
- 公式サイトのURLを記載
- 免責事項を必ず入れる

### ステップ5: その他の設定

**SNSリンク**（オプション）:
- Twitter/X
- Facebook
- Instagram
- YouTube

今はなくてもOK。今後、SNS運用を始めたら追加しましょう。

**外部サイトURL**:
- BoatAI公式サイトのURL: `https://boat-ai.jp/`

### ステップ6: 保存

すべて入力したら「保存する」をクリック。

## 3. 記事の投稿方法

プロフィール設定が完了したら、最初の記事を投稿しましょう。

### ステップ1: 記事作成画面を開く

1. 右上の「投稿」ボタンをクリック
2. 「テキスト」を選択（通常の記事）

### ステップ2: タイトルを入力

準備した記事のタイトルをコピー&ペースト。

**例**: `BoatAI - AIが競艇予想を無料で公開。使い方ガイド`

### ステップ3: 本文を入力

準備したMarkdownファイルの内容をコピー&ペースト。

**注意点**:
- noteは**見出し記法**をサポートしています
- `# 見出し1` → 大見出し
- `## 見出し2` → 中見出し
- `### 見出し3` → 小見出し
- 表、リスト、太字なども対応

**Markdownから変換する方法**:
1. Markdownファイルの内容をそのままコピー
2. noteのエディタにペースト
3. noteが自動的にフォーマットしてくれます

**もし表示が崩れたら**:
- 見出しを手動で調整
- 太字: `**太字**` または選択してBボタン
- リスト: `-` または `1.` で始める

### ステップ4: アイキャッチ画像を設定（オプション）

記事のトップに表示される画像。

**推奨画像**:
- サイズ: 横長（推奨: 1280×670px）
- 内容: BoatAIのスクリーンショット、競艇場の写真など

**設定方法**:
1. 本文の上にある「+」ボタンをクリック
2. 「画像」を選択
3. 画像をアップロード

**画像がない場合**:
- 初期は設定しなくてもOK
- テキストのみでも問題なし

### ステップ5: ハッシュタグを追加

記事の最後に以下のハッシュタグを追加してください：

```
#競艇 #ボートレース #AI予想 #BoatAI #データ分析
```

**ハッシュタグの効果**:
- 検索されやすくなる
- 関連記事として表示される
- 読者の流入増加

### ステップ6: 公開設定

**公開範囲**:
- **全体に公開**（推奨）
  - 誰でも閲覧可能
  - Google検索にも表示される
  - AdSense申請には必須

**有料記事にはしない**:
- AdSense申請には「無料コンテンツ」が必要
- 初期は全て無料公開が推奨

**コメント設定**:
- 「コメントを受け付ける」にチェック（推奨）
- 読者とのコミュニケーションが可能

### ステップ7: プレビューで確認

「プレビュー」ボタンで表示を確認します。

**チェックポイント**:
- タイトルは正しいか
- 見出しが適切に表示されているか
- 表やリストが崩れていないか
- ハッシュタグが入っているか
- URLリンクが機能しているか

### ステップ8: 公開

問題なければ「公開する」ボタンをクリック。

**公開後の確認**:
- 自分のnoteページで記事が表示されるか確認
- リンクをシェアできるか確認

## 4. Google AdSense申請を意識した運用

将来的にAdSense申請を行うための注意点です。

### 4-1. コンテンツの質

**必須事項**:
- オリジナルコンテンツ（コピペNG）
- 有益な情報を提供
- 1記事1000文字以上（推奨: 2000文字以上）
- 誤字脱字をチェック

**今回の記事の文字数**:
- 記事1: 約3,000文字 ✅
- 記事2: 約3,500文字 ✅
- 記事3: 約4,000文字 ✅
- 記事4: 約3,800文字 ✅
- 記事5: 約4,200文字 ✅

→ すべて基準をクリアしています。

### 4-2. 投稿頻度

**推奨スケジュール**:
- 週1〜2回の投稿
- 定期的な更新が重要
- 一気に全部投稿せず、分散させる

**投稿スケジュール例**:
- 12月17日（火）: 記事1「BoatAI紹介・使い方ガイド」
- 12月19日（木）: 記事2「的中実績レポート」
- 12月21日（土）: 記事3「競艇データ分析の基礎知識」
- 12月24日（火）: 記事4「AIによる予想の仕組み解説」
- 12月26日（木）: 記事5「競艇場別の特徴と攻略法」

### 4-3. ギャンブルコンテンツの注意点

**AdSenseのポリシー**:
- ギャンブル関連コンテンツは制限される場合がある
- 教育的・情報提供的な内容ならOK
- 「必ず勝てる」などの誇大広告はNG

**対策**:
- **データ分析の教育的コンテンツ**として位置づける ✅
- 「的中を保証するものではない」という免責事項を必ず記載 ✅
- 統計やデータ分析の解説を中心に ✅
- 舟券購入の直接的な勧誘は避ける ✅

### 4-4. プロフィールの充実

**必須事項**:
- プロフィール画像を設定 ✅
- 自己紹介文を充実させる ✅
- 外部サイトリンク（BoatAI公式サイト）✅

### 4-5. 継続的な更新

**AdSense申請前に必要なこと**:
- 記事数: 10記事以上（推奨: 15〜20記事）
- 運用期間: 1〜2ヶ月以上
- 定期的な更新

**今後の記事案**:
- 毎週の実績レポート（継続コンテンツ）
- 月次まとめ
- 特定の競艇場の詳細分析
- 選手データの見方
- 風・波の影響解説
