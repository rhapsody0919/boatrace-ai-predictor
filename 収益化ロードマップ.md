## 🛡️ 6. リスク管理

### ⚠️ 法的リスク
```
✅ 対策:
- 「必ず勝てる」等の断定表現は使わない
- 「参考情報の提供」と明記
- 賭博の勧誘・代理購入は一切行わない
- 利用規約・免責事項を明記
```

### ⚠️ AdSense停止リスク
```
✅ 対策:
- ポリシー遵守（ギャンブル関連コンテンツのガイドライン確認）
- クリック誘導禁止
- 無効なトラフィック対策
- 定期的なポリシー確認
```

### ⚠️ スクレイピング停止リスク
```
✅ 対策:
- robots.txt遵守
- アクセス頻度制限（1秒に1リクエスト以下）
- User-Agent明記
- 公式API検討（将来的に）
```
