# モーター性能で勝率が変わる理由【2連対率40%超えを狙え】

## はじめに

「A1級の選手が1号艇なのに負けた...なぜ？」

その答えは**モーター**にあります。

ボートレースでは、選手は抽選でモーターを割り当てられます。このモーター性能の差が、選手の実力以上にレース結果を左右するのです。

この記事では、**モーター性能の見方**と**勝てるモーターの見抜き方**を徹底解説します。

---

## モーターとは？なぜ重要なのか

### モーターの基本

**モーターの仕組み:**
- 全選手が使用するエンジンは統一規格（ヤマハ製）
- しかし、個体差がある（新品・整備状況・運による当たり外れ）
- 選手は前検日に**抽選**でモーターを引く

**使用期間:**
- 1つのモーターは約1年間使用
- その間、複数の選手が共用
- 整備は選手が自分で行う

**重要なポイント:**
モーター性能が良い = 選手が整備上手 + 元々の当たりモーター

---

## モーター2連対率の見方

### 2連対率とは

**定義:**
そのモーターを使った過去のレースで、1-2着に入った割合

**計算式:**
```
2連対率 = （1着回数 + 2着回数）÷ 出走回数 × 100
```

**例:**
```
モーター No.25
出走回数: 100回
1着: 25回
2着: 15回
→ 2連対率 = 40%
```

---

### 数値の評価基準

| 2連対率 | 評価 | 信頼度 |
|---------|------|--------|
| **45%以上** | 超優良モーター | ⭐⭐⭐⭐⭐ |
| **40-45%** | 優良モーター | ⭐⭐⭐⭐ |
| **35-40%** | 普通 | ⭐⭐⭐ |
| **30-35%** | やや不調 | ⭐⭐ |
| **30%未満** | 不調モーター | ⭐ |

**目安:**
- 40%以上: 積極的に狙える
- 35%未満: 警戒が必要

---

## モーター性能が勝敗を左右する理由

### 理由1: 出力差が直接スピードに影響

**モーターの役割:**
- ボートを推進する唯一の動力源
- 出力が高い = 加速力・最高速度が高い

**性能差の実例:**

```
優良モーター（2連対率42%）:
- 展示タイム: 6.75秒
- 直線で他艇を引き離す

不調モーター（2連対率28%）:
- 展示タイム: 6.92秒
- 直線で置いていかれる
```

**0.1秒の差**が、着順に大きく影響します。

---

### 理由2: ターン性能（まわり足）

**まわり足とは:**
ターン時の安定性と速さ

**良いモーター:**
- スムーズにターンできる
- ターン後の加速が速い
- インから抜かれにくい

**悪いモーター:**
- ターンで外に膨らむ
- ターン後の加速が遅い
- 外側に抜かれやすい

**ボートレースの特性上:**
- ターンは1周2回
- ターン性能の差が積み重なる
- 3周で6回のターン = 差が顕著に

---

### 理由3: 選手の実力を引き出せるか

**重要な考え方:**
選手の実力 × モーター性能 = 総合力

**実例比較:**

```
【パターンA】
A1級選手（勝率7.5） × 優良モーター（45%）
→ 総合力: 最強 → 勝率70%

【パターンB】
A1級選手（勝率7.5） × 不調モーター（28%）
→ 総合力: 中程度 → 勝率40%

【パターンC】
B1級選手（勝率5.5） × 優良モーター（45%）
→ 総合力: 中の上 → 勝率30%

【パターンD】
B1級選手（勝率5.5） × 不調モーター（28%）
→ 総合力: 最弱 → 勝率10%
```

**結論:**
どれだけ実力がある選手でも、モーターが悪いと勝てない

## 優良モーターの見抜き方

### 1. 出走表の2連対率をチェック

**確認場所:**
出走表の「モーター」欄

**見方:**
```
モーター: No.25
2連対率: 42.5%
```

**判断基準:**
- 40%以上 → 即買い候補
- 35-40% → 様子見
- 35%未満 → 警戒

---

### 2. 展示航走で最終確認

**展示タイムを見る:**
- 6.75秒以下: 優秀
- 6.75-6.85秒: 普通
- 6.85秒以上: やや遅い

**まわり足を目視:**
- スムーズなターン → ○
- 外に膨らむ → ×

**伸び足を確認:**
- 直線でグングン加速 → ○
- 伸びが鈍い → ×

---

### 3. 過去の使用選手をチェック

**ポイント:**
同じモーターを使った過去の選手の成績を見る

**確認方法:**
ボートレース公式サイト → モーター成績 → 使用選手一覧

**実例:**

```
モーター No.25（2連対率42%）

過去の使用選手:
- A1級 山田選手: 1-2-1-1-3（優秀な成績）
- A2級 佐藤選手: 2-1-4-2-1（好走）
→ 誰が乗っても好成績 = 優良モーター確定
```

---

## モーター性能の落とし穴

### 注意点1: 新品モーターの評価

**問題:**
- 交換直後のモーターは、まだデータが少ない
- 2連対率が正確でない

**対策:**
- 使用節数が3節以上のデータを重視
- 1-2節目は展示航走で判断

---

### 注意点2: 選手の整備力

**重要な事実:**
モーター整備は選手の技術に依存

**例:**

```
モーター No.30（2連対率35%）

前半3節（A1級選手が使用）:
2連対率 45%（優良）

後半3節（B2級選手が使用）:
2連対率 25%（不調）

→ 平均すると35%だが、実際は整備力次第
```

**対策:**
今節の担当選手の整備技術を確認
- A1級 → 整備技術も高い傾向
- B2級 → 整備が不十分な可能性

---

### 注意点3: 場による得意不得意

**モーターにも相性がある:**
- 静水面（芦屋、大村）に強いモーター
- 荒れた水面（戸田、江戸川）に強いモーター

**対策:**
当地での成績を重視
