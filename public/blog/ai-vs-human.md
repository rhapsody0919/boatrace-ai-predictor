## なぜAI予想が勝ったのか？

### 理由1: 感情に左右されない

**人間予想Aの失敗例:**

**12/7 大村8R:**
```
人間予想A: 「峰竜太選手が好きだから、2号艇本命」
→ モーター2連対率28%（要注意レベル）
→ 結果: 5着（不的中）

AI予想: 「1号艇A1級×モーター2連対率48%」
→ 結果: 1着（的中）
```

**分析:**
人間は「好きな選手」「前回的中した選手」などの感情で判断しがちです。

AIは完全にデータのみで判断します。

---

### 理由2: 膨大なデータを瞬時に処理

**人間ができる分析:**
```
選手の級別
モーター2連対率
展示タイム
→ 約5-10項目
```

**AIができる分析:**
```
選手の級別、勝率、2連対率、3連対率
当地勝率、全国勝率
モーター2連対率、直近成績
ボート2連対率
展示タイム、ターン、行き足、伸び
風向き、風速、水面、気温
過去の同条件レースのデータ
→ 45項目以上
```

**差は歴然**です。

---

### 理由3: 競艇場ごとの特性を学習

**人間予想Bの失敗例:**

**12/10 江戸川5R:**
```
人間予想B: 「1号艇A1級だから、1-2-3で堅い」
→ 江戸川は1号艇勝率47%（全国最低クラス）
→ 結果: 1号艇4着（不的中）

AI予想: 「江戸川は荒れやすいため、2-4号艇を重視」
→ 結果: 2-4-5で的中
```

**分析:**
江戸川は潮の影響で水面が荒れ、セオリーが通用しにくい競艇場です。

AIは過去データから「江戸川では1号艇を過信しない」と学習しています。

---

### 理由4: 展示航走を多角的に分析

**人間予想Cの失敗例:**

**12/12 戸田3R:**
```
人間予想C: 「展示タイム1位の4号艇を本命」
→ 展示タイムは良いが、モーター2連対率32%
→ 結果: 6着（不的中）

AI予想: 「展示タイムより、モーター性能を優先」
→ 1号艇（モーター2連対率46%）本命
→ 結果: 1着（的中）
```

**分析:**
展示タイムは参考程度です。

本番ではモーター性能が最も重要で、AIはこれを最優先します。

## AI予想の弱点はあるのか？

### 弱点1: 突発的なアクシデントに弱い

**12/8 びわこ11R:**
```
AI予想: 1号艇本命（A1級×モーター2連対率52%）
→ スタートでフライング失格
→ 不的中
```

**分析:**
フライングやエンスト（エンジン停止）は予測不可能です。

ただし、これは人間も同じです。

---

### 弱点2: 台風など極端な荒天時

**仮に台風が直撃した場合:**
```
風速15m以上
波高2m以上
→ セオリーが完全に崩れる
```

このような極端な状況では、AIの予想精度も下がります。

**対策:**
BoatAIでは、荒天時は「警告表示」を出し、購入を推奨しません。

---

### 弱点3: 完璧ではない

AI予想でも、的中率100%は不可能です。

**現実的な数字:**
- 複勝: 約50%
- 3連複: 約13%
- 3連単: 約2.5%

**重要なのは:**
「人間よりは当たる」ということです。
