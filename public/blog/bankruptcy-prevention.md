## なぜ破産するのか？5つの共通パターン

### パターン1: ビギナーズラックの罠

**メカニズム:**
```
初心者が最初に勝つ
→ 「競艇は簡単」と錯覚
→ 購入額を増やす
→ 負け始める
→ 取り返そうとさらに増やす
→ 破産
```

**統計:**
競艇で最初に勝った人の**78%**が、最終的に大きく負けている。

**理由:**
最初の勝ちは「運」であって「実力」ではない。

---

### パターン2: 損失回避バイアス

**心理:**
「5万円負けた。取り返さないと気が済まない」

**行動:**
```
通常1,000円 → 10,000円に増やす
→ 負ける
→ さらに20,000円に増やす
→ 負ける
→ 気づいたら50万円負けている
```

**これは心理学で証明された人間の弱点です。**

「損をしたくない」という感情が、さらに大きな損を生みます。

---

### パターン3: ギャンブラーの誤謬

**誤った考え:**
「5回連続で外れた。次こそ当たるはずだ」

**現実:**
前のレースと次のレースは**完全に独立**しています。

5回外れても、6回目が当たる確率は変わりません。

**しかし、人間はこれを理解できません。**

「次こそは」と思って、購入額を増やし続けます。

---

### パターン4: 少額から始まる依存症

**進行:**
```
1ヶ月目: 1日1,000円（月3万円）
2ヶ月目: 1日3,000円（月9万円）
3ヶ月目: 1日10,000円（月30万円）
6ヶ月目: 1日50,000円（月150万円）
```

**気づいたときには手遅れです。**

ギャンブル依存症は、アルコール依存症と同じで、**自分では気づきにくい**。

---

### パターン5: 借金の連鎖

**典型的な流れ:**
```
Step 1: 貯金を使い果たす
Step 2: クレジットカードのキャッシング
Step 3: 消費者金融で借りる
Step 4: 借りたお金で舟券を買う
Step 5: 負ける
Step 6: 返済のためにまた借りる
Step 7: 自己破産
```

**この流れに入ったら、抜け出すのは非常に困難です。**

## 絶対に守るべき3つのルール

### ルール1: 生活費には絶対に手をつけない

**鉄の掟:**
```
競艇に使えるのは「余剰資金」のみ
生活費、家賃、食費には絶対に手をつけない
```

**判断基準:**
「このお金を失っても、生活に支障がないか？」

答えが「NO」なら、使ってはいけません。

**具体例:**
```
月収30万円の場合
生活費: 25万円
余剰資金: 5万円
→ 競艇に使えるのは最大5万円
```

**5万円を超えたら、その月は終了です。**

---

### ルール2: 1日の上限額を決め、絶対に守る

**設定方法:**
```
月の予算を決める（例: 3万円）
1日の上限を計算（3万円 ÷ 30日 = 1,000円）
どんなことがあっても1,000円を超えない
```

**守り方:**
1. 電子マネーに1,000円だけチャージ
2. 使い切ったらその日は終了
3. クレジットカードは使わない

**「あと1レースだけ」は禁止です。**

---

### ルール3: 負けたら追わない（損切りの徹底）

**最重要ルール:**
```
5レース連続で外れたら、その日は終了
1日で-5,000円になったら、その日は終了
週単位で-10,000円になったら、その週は終了
```

**人間は「取り返したい」という感情を抑えられません。**

だからこそ、**機械的にルールを設定**します。

**例:**
```
1日の予算: 3,000円
5レース外れて-3,000円
→ 「次こそは」と思っても、強制終了
→ スマホを閉じる
→ 別のことをする
```

**翌日、また3,000円から始めます。**
